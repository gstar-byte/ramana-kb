## 侧边栏折叠按钮修复（20260419-6）

### 问题
多个书籍章节页面的侧边栏折叠/展开按钮（▶/▼）无法点击：
- 《面对面》5个章节页面
- 《走向静默如你本来》部分章节页面
- 《日日与彼》部分章节页面

### 原因
页面内联脚本与 `script.js` 中的事件绑定冲突：
```javascript
// 内联脚本（先执行）
document.querySelectorAll('.sidebar-section-title').forEach(title => {
    title.addEventListener('click', () => title.parentElement.classList.toggle('collapsed'));
});

// script.js（后执行）
document.querySelectorAll('.sidebar-section-title').forEach(function(title){
    title.addEventListener('click', function(){ toggleSection(this); });
});
```

两次绑定导致冲突，且内联脚本的简单切换与 `toggleSection` 函数的实现不一致。

### 修复方案
移除所有章节页面内联脚本中的 `sidebar-section-title` 事件绑定，只保留 `script.js` 的统一处理。
