# 工作日志 2026-04-13

## Astro架构迁移

### 项目概述
将拉玛那马哈希知识库从纯HTML重构为Astro静态站点生成器。

### 新项目结构
```
ramana-astro/
├── src/
│   ├── components/
│   │   └── Sidebar.astro     ← 全局侧边栏组件
│   ├── layouts/
│   │   └── BaseLayout.astro  ← 基础布局模板
│   └── pages/
│       ├── index.astro
│       ├── graph.astro
│       ├── books/
│       │   ├── index.astro
│       │   └── be-as-you-are.astro
│       ├── concepts/
│       │   ├── index.astro
│       │   └── atman.astro
│       ├── persons/
│       │   ├── index.astro
│       │   └── ramana.astro
│       └── methods/
│           └── index.astro
├── public/
│   └── styles/
│       └── global.css         ← 全局样式
└── dist/                     ← 构建输出（纯静态HTML）
```

### 架构优势
1. **组件化** - 侧边栏只需改一个文件，全站生效
2. **SEO友好** - 构建时生成完整HTML，爬虫可读
3. **Vercel部署** - `npm run build` 后 `dist/` 目录直接部署

### 关键文件
- `Sidebar.astro` - 全局侧边栏组件
- `BaseLayout.astro` - 基础布局
- `global.css` - 全局样式
