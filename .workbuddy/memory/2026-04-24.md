# 2026-04-24 工作日志

## 侧边栏桌面端开合修复（重大调试）

### 问题1：首页和图谱页按钮不工作
- **根因**：内联 `<script>` 中的事件监听器和 `defer` 的 `script.js` 中的事件监听器重复注册，导致点击一次触发两次 `toggleSidebarDesktop()`（添加 `desktop-hidden` → 又立即移除），状态瞬间还原
- **解决**：从 `script.js` 中删除 `toggleSidebarDesktop` 函数和事件监听器，所有页面统一用内联脚本

### 问题2：关闭侧边栏后内容区域没有变宽
- **根因**：`.content-wrapper` 和 `<main>` 有 `max-width:var(--content-max)`（1100px），即使 `.main-content` 的 `margin-left` 变为0，内容容器仍被限制
- **解决**：在内联 CSS 和 styles.css 末尾添加 `.main-content.full-width .content-wrapper,.main-content.full-width main{max-width:100% !important;}`

### 教训
- 不要在 defer 外部 JS 和内联 `<script>` 中重复注册相同的事件监听器
- 修改布局宽度时，需要同时考虑 `.main-content` 和内部的 `.content-wrapper` / `<main>` 容器
