## 图谱移动端交互升级（graph.html）

### 需求
移动端点击节点因为没有鼠标悬停效果，会直接跳转，看不到关联线。

### 解决方案
- **短按（< 600ms）**：显示关联线 + tooltip，3秒后自动跳转详情页
- **长按（>= 600ms）**：仅显示关联线查看，不跳转，手抬起后3秒自动隐藏
- **PC端**：保持鼠标 hover 显示关联、click 跳转的原有逻辑
- 新增 `isTouchDevice()` 工具函数区分PC/移动端，mouseover/click 事件在移动端直接 return

### 顶部说明文字
- PC：`🖱️ 鼠标悬停节点查看关联线，单击跳转详情页`
- 移动：`👆 轻触节点显示关联线（3秒后自动跳转详情页）；长按仅查看关联不跳转`
